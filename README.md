# My Movie Collection

Personal movie collection manager.

## Features:

- Add movies
- Delete Movies
- Update Movies
- Search Movies by category
- Storage managed in local storage.

## Built With

* React.js
* Material UI
* Node
* Browserify
* Express
* ESLint (AirBnB)
* Heroku

## Getting Started

Navigate to [MyMovieCollection](https://mymoviecollection.herokuapp.com/)

OR

## Install locally

Fork or clone code

```
npm install
```
After dependencies finish installation

```
npm start
```

In your browser, navigate to localhost:8000.

* Enjoy the app!
